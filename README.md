# Sales Data Analysis

## Project Objective
Analyze sales data to extract insights about product categories, cities, and shipping times.  
The goal is to demonstrate data cleaning, analysis, and visualization skills.

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Dataset
Superstore sales dataset containing 9800 orders with columns like Order Date, Ship Date, Category, Sub-Category, Sales, etc.

## Key Insights
- **Total Sales:** 2261536.7827000003
- **Highest Selling Category:** Technology 
- **Average Shipping Days:** 4
